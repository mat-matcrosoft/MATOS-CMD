# MatOS-CMD[Мэ'т ОС] 
**MatOS-CMD** Командная строка разработанная на C++ для платформ Windows и Linux(в ближайшее время).

Mat-CMD это мощный терминал для выполнения задач от базовых действий до написания запуска кода внутри приложения!

# Системные требования.
 > # `System - Windows XP/7/8/10/11.`
 > # `Разрядность системы - x86 / x64.`
 > # `Клавиатура.`

 # Почему Mat-CMD удобно?
> Данное програмное обеспеченье разработано Matcrosoft Software. Командная строка позволяет выполнять различные задачи в одном приложении.
> В версии от 2.0.2 будет реализована вохможность дополнения интерфейса командной строки различными командами.
